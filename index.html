This is an exercise Portfolio of Interactive Data Visualization class which is designed to introduce interactive data visualization concepts in a web environment. I am using d3 library to create charts and interactive charts.

Introduction

  1. The bar chart is about This data set represents the 2018 league-wide standings in Major League Baseball. This includes additional data related to scoring and record splits. It is presented using comma-separated values (csv).

  2. The scatterplot is about This data set represents 2018 National Football League rushing statistics as it relates to players who are either running backs (RB) or fullbacks (FB). It is presented using comma-separated values (csv).

  3. This line chart provides monthly stock information for Apple Inc. (AAPL) from Jan. 1, 1996 to March 1, 2019.

  4. This area chart provides monthly stock information for Amazon from May 1, 1997 to March 1, 2019.

  5. This interactive line chart shows monthly stock information for Walt Disney Company.

  6. This choropleth Map shows total veteran percentage in the United States.

  7. This interactive map shows the locations of the Seven Ancient and Modern Wonders of the world.
